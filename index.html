Project Title: Personal Blog

This project involves hosting a simple personal blog website that allows users to read posts and interact with content online. The aim is to learn and demonstrate how to deploy a live website using Amazon EC2 infrastructure. This project compares cloud hosting models and practices related to cost-effectiveness, ease of deployment, and licensing decisions. The site is hosted using Ubuntu on a t2.micro EC2 instance and served with Apache2.

The blog will have basic content structure and focuses more on the deployment process rather than complex features. It reflects practical application of cloud concepts from ICT171 and prepares me for real-world hosting and license usage.
